# Sets up a new Mac

## What this script does

1. This script will install the following:
  - Homebrew
  - Google Chrome
  - Nextcloud
  - OnlyOffice
  - OpenOffice
  - Git
2. Adds a new Wifi network
3. Adds a new user

## How to run this script

```sh
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/ValentinKolb/mac-setup/main/setup.sh)"
````
